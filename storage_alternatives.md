# بدائل Firebase Storage المجانية للمشروع

بناءً على طلبك، إليك ملخص للبدائل التي توفر مساحة أكبر من 5 جيجابايت وسهلة التكامل مع Next.js:

## 1. Cloudinary (الخيار الموصى به)
- **المساحة:** ~25 جيجابايت.
- **الاستخدام:** ممتاز للصور والفيديو، يدعم التصغير التلقائي (Optimization).
- **طريقة الربط:** 
  1. تثبيت المكتبة: `npm install next-cloudinary`
  2. إضافة مفاتيح API في `.env`.
  3. استخدام مكوّن `<CldImage />` لعرض الصور.

## 2. Cloudflare R2
- **المساحة:** 10 جيجابايت مجانية.
- **المميزات:** لا توجد رسوم على التحميل (Egress)، متوافق مع S3.
- **طريقة الربط:** يتطلب مكتبة `@aws-sdk/client-s3`.

## 3. Backblaze B2
- **المساحة:** 10 جيجابايت مجانية.
- **المميزات:** سهل جداً في التعامل مع الملفات الكبيرة.

## 4. خيار الحفظ في قاعدة البيانات (Prisma)
- **الطريقة:** استخدام نوع `Bytes` في Prisma.
- **ملاحظة:** لا ننصح به للصور الكبيرة، فقط للأيقونات أو الصور الصغيرة جداً.
